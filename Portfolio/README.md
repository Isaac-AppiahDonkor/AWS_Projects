🌐 AWS Static Portfolio Website Hosting
A beginner-friendly AWS cloud project demonstrating how to deploy a static portfolio website using Amazon S3 and Amazon CloudFront. This project was completed as part of my cloud engineering learning journey to gain practical experience with AWS core services and static website deployment.

📌 Project Overview
This project showcases how to host a static website on AWS using Amazon S3 and deliver it globally through Amazon CloudFront.
The primary goal was to understand how different AWS services work together to provide a scalable, reliable, and cost-effective web hosting solution.
