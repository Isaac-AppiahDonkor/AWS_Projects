PUT YOUR IMAGES HERE
====================

Drop your images in this folder with these exact names:

  image1.jpg  →  your profile photo
  image2.jpg  →  project 1 screenshot (S3 portfolio)
  image3.jpg  →  project 2 screenshot (EC2 web app)
  image4.jpg  →  project 3 screenshot (VPC + RDS)
  image5.jpg  →  project 4 screenshot (Lambda resizer)
  image6.jpg  →  project 5 screenshot (auto-scaling)
  image7.jpg  →  project 6 screenshot (AI capstone)

Until you add an image, the site shows a friendly emoji placeholder.
So you can host it today and add screenshots as you finish each project.

You can use .png instead of .jpg — just update the filename in config.json.

Recommended sizes:
  - Profile photo: square, at least 400x400px
  - Project screenshots: 16:9 ratio, at least 1200x675px
